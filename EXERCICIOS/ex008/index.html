Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântico)

E nessa frase, usamos o termo em destaque usando a tag strong (semantica)

Itálico / Enfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica)

E nessa frase, temos o termo em itálico usando a tag EM (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Podemos alterar a cor do texto marcado usando 'style'

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Ou podemos usar a não semântica para marcar um texto com inserido

Texto sobreescrito

Para inserir coisas como x20+3

Textos subscritos

Para inserir coisas como H20
